Plots x and y on ax

Parameters

  • ax (matplotlib.pyplot.Axes object) – Axes where to plot

  • x (ndarray) – abcissa values

  • y (ndarray) – ordinate values

  • plot (string, optional (default: 'plot')) – string specifying the plot function to be returned: ‘plot’ - Axes.plot ‘semilogy’ - Axes.semilogy ‘semilogx’ - Axes.semilogx ‘loglog’ - Axes.loglog

  • nuclei (boolean, optional (default: True)) – switch for using the total number of particles (False) or nuclei (True) as the total of the system: only relevant when plotting densities

  • linewidth (float, optional (default: 2)) – default linewidth of plot

  • plottot (boolean, optional (default: True)) – show the total along axis=0

  • ylabel (string, optional (default: r'[$ 10^{{{}}}$]')) – the y-axis label: attempts to format the decade into the ylabel

  • totlabel (string, optional (default: 'Total')) – Label for the total-line, if plotted

  • plotmax (int, optional (default: 1000)) – the maximum numbers of sets from y to be plotted

Returns

Return type

None
